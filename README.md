# Ren'Py 文档索引（Ren'Py docsup）

> 使用镜像的方式为 Ren'Py 项目的 `python`  块提供文档字符串索引，支持函数/变量的 **文档预览** 和 **跳转到定义**。

## ✨ 功能

* **Hover**
  鼠标悬停在Ren'Py的 Python 代码上，显示定义
* **跳转到定义**
  ps:可能存在部分问题
* **类型推断**
  只能显示第一次定义时的类型
