# Ren'Py 文档索引（Ren'Py docsup）

> 使用镜像的方法为 Ren'Py 项目的 `python:` 和 `init python:` 块提供文档字符串索引
> ps.短期内应该不更新了，有问题我会修

## ✨ 功能

![1765489036580](images/README/1765489036580.png)

- **Hover**
  悬停在 Python 代码上，自动显示定义（当前不支持跨文件）
  ps.对类的调用可能会和官方支持”打架“，类还是用官方的好，本扩展当前只能对同一个rpy文件下的类有效
- **重建索引**
  命令面板执行 **“renpy重建索引”**
- **跳转到定义**
  ps:0.0.4版不可用
